Разбирает строку времени в структуру Time

Аргументы
time_strСтрока в формате "HH:MM" (24-часовой формат)
Возвращает
Структура Time с разобранными часами и минутами
Заметки
Входная строка должна строго соответствовать формату "HH:MM"
Предупреждения
Не выполняет валидацию ввода - некорректный ввод приводит к неопределённому поведению

Time t = parse_time("14:30"); // t будет содержать {14, 30}
