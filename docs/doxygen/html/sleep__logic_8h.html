Подробное описание

Заголовочный файл для модуля анализа сна

Модуль предоставляет функции для анализа продолжительности и качества сна на основе времени отхода ко сну и пробуждения.

Функции

◆ calculate_sleep()

SleepResult calculate_sleep ( SleepData  data)

Основная функция анализа сна

Рассчитывает продолжительность сна на основе входных данных и дает текстовую оценку его качества. Учитывает случаи сна через полночь.

Аргументы
dataВходные данные о времени сна (SleepData)
Возвращает
SleepResult Структура с результатами анализа
Заметки
Функция автоматически обрабатывает переход через полночь
Предупреждения
Для корректной работы требуется подключенный модуль time_utils.h

Пример использования:

SleepData data = {{22, 30}, {6, 45}};
SleepResult result = calculate_sleep(data);
printf("Спали %d часов %d минут: %s\n",
result.hours, result.minutes, result.comment);
