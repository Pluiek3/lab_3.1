Time minutes_to_time ( int  total_minutes)

Конвертирует минуты в структуру Time.

Аргументы
total_minutesОбщее количество минут
Возвращает
Time Структура с часами и минутами
Заметки
Выполняет нормализацию часов (значение часов будет в диапазоне 0-23)

Time parse_time ( const char *  time_str)

Парсит строку времени в формате HH:MM в структуру Time.

Аргументы
time_strСтрока времени в формате "HH:MM"
Возвращает
Time Структура с распарсенным временем
Заметки
Не выполняет валидацию формата входной строки
Предупреждения
При неверном формате строки поведение не определено

Пример использования:

Time t = parse_time("23:45");
printf("%02d:%02d", t.hours, t.minutes); // Выведет "23:45"
